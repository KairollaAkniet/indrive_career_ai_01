# inVision U AI Ecosystem — Selector for inDrive

Интеллектуальная система для автоматизации отбора кандидатов в университет **inVision U**. Проект представляет собой связку **Telegram-бота** (для AI-интервью) и **Web-платформы** (для финальной регистрации).


## Описание
Система анализирует текстовые заявки абитуриентов, оценивая их лидерский потенциал, социальные инициативы и мотивацию расти в IT. 
- **Бот** выступает первичным фильтром.
- **Сайт** служит платформой для глубокого ознакомления с программой.

## Технологии
- **Язык:** Python 3.10+
- **Библиотека:** aiogram 3.x
- **AI Core:** Hugging Face Inference API (Model: Meta-Llama-3)
- **Frontend:** HTML5, CSS3, JavaScript (разработка партнера по команде)

## Архитектура
1. Кандидат проходит опрос в Telegram.
2. AI-движок вычисляет `Score` на основе соответствия ценностям inDrive.
3. Бот перенаправляет успешно прошедших этап на Web-сайт для завершения регистрации.

## Как запустить
1. Склонируйте репозиторий.
2. Создайте файл `.env` (не для GitHub) и добавьте:
   `BOT_TOKEN=ваш_токен`
   `INDRIVE_API_KEY=ваш_ключ`
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите: `python main.py`.
