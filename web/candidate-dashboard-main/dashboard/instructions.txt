## Инструкции по запуску (Windows / PowerShell)

### 1) Бэкенд (FastAPI, порт `8000`)
1. Откройте PowerShell в корне проекта (папка `Cursor`):
   - `cd "c:/Users/baxa/Downloads/Cursor"`
2. Создайте виртуальное окружение и установите зависимости:
   - `cd backend`
   - `python -m venv venv`
   - `.\venv\Scripts\Activate.ps1`
   - `pip install -r requirements.txt`
3. Вернитесь в корень проекта:
   - `cd ..`
4. Заполните БД тестовыми кандидатами:
   - `python -m backend.seed_data`
5. Запустите сервер:
   - `uvicorn backend.main:app --reload --port 8000`

Бэкенд будет доступен по адресу:
- http://localhost:8000

### 2) Фронтенд (Vite, порт `5173`)
1. Откройте отдельный PowerShell:
   - `cd "c:/Users/baxa/Downloads/Cursor/frontend"`
2. Установите зависимости:
   - `npm install`
3. Запустите dev-сервер:
   - `npm run dev`

Фронтенд будет доступен по адресу:
- http://localhost:5173

### Важно по CORS
В бэкенде включён CORS, чтобы фронтенд с `http://localhost:5173` мог обращаться к API на `http://localhost:8000`.
